# exercise_git_rebase

Esse repositório contém o exercicio de Git Rebase

Nesse exercício iremos praticar o `git rebase`. A situação é a seguinte:

Existia a branch `main` o aluno queria incluir uma nova feature nesse repostiório que exigia uma pasta `cpp/` que ainda não existia. Ele criou uma nova branch chamada `aluno/feat/cpp-hello-world` e criou a nova pasta com os códigos de cpp que ele havia feito.

Depois de um tempo, a `main` foi atualizada com a inclusão da pasta `cpp/` e com o arquivo `README.md` dela e também com o commit atualizando o README principal.

Seu objetivo é realizar o rebase da branch `aluno/feat/cpp-hello-world` sobre a `main` mais atualizada.
